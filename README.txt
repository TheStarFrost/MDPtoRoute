## 3. 算法运行
Route.cpp编译后即可运行
编译指令为 `g++ Route.cpp -o Route`
运行指令为 `./Route`
若选择手动输出模式,则需要注释掉Route.cpp的103行,并在运行时按照提示输入数据
若选择自动输入模式,则可在本文件同级文件夹下,找到input.txt文件
在其中输入和手动输入格式一致的数据,即可自动运行
此方法需要在Route.cpp文件中关掉103行的注释,并忽略输入提示信息

input.txt文件中的数据格式如下:
n                          路由器数量
0->0 0->1 ... 0->n-1       0号路由器到其他路由器的开销(0->0表示0号路由器到0号路由器的开销,下同,不再另行解释)
...                        ...号路由器到其他路由器的开销
n-1->0 n-1->1 ... n-1->n-1 n-1号路由器到其他路由器的开销

与说明同级的input.txt文件中数据为大作业报告中的数据,可参照处理

## 4. 算法结果
